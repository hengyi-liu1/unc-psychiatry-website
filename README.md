# Account Management System for UNC Psychiatry Website

Full-stack account management system developed for the UNC Department of Psychiatry website. Features include invitation-based account creation, secure authentication, password reset functionality, and responsive user interfaces.

## Demo Video

[![Short demo video](https://img.youtube.com/vi/Ml1vxDOrQmA/0.jpg)](https://youtu.be/Ml1vxDOrQmA)

## Key Features

- Developed a responsive sign-up and authentication workflow using Angular and Material Design components.
- Implemented secure invitation-based account registration to protect patient privacy and restrict unauthorized access.
- Integrated MongoDB storage for invitation tokens, user account data, and securely hashed passwords.

## Technical Skills

- Angular, TypeScript
- Express, Node.js
- Mongoose, MongoDB
- Git, Docker
- Agile Workflow
